# EGE_Patterns - Список шаблонов для ЕГЭ по информатике 2026

Содержит файлы с объяснениями решения задач, также только исходный код без объяснений.

Весь код основан на основных прототипах, представленных в <strong>Демоверсии ЕГЭ по Информатике 2026</strong>(вариант на [kompege.ru](https://kompege.ru/variant?kim=25107025) или [4ege.ru](https://4ege.ru/informatika/76077-demoversija-ege-2026-po-informatike.html)).

Также для шаблонов с объяснениями представлены и комментарии по поводу самого задания, лайфхаки и вариации решений(может быть несколько способов решения задачи)
### Структура проекта
```
EGE_Patterns/
├── patterns_with_explanations/   # Шаблоны с объяснением каждой строки кода
│   ├── 1.py                      # Файлы с номерами заданий (внутри может быть несколько прототипов)
│   └── 2.py
│       ...
│
├── only_patterns/                # Те же шаблоны, только без объяснений
│   ├── 1.py
│   └── 2.py
│       ...
└── README.md                     # Этот файл
```

Все приведенные шаблоны написаны на <strong>Python</strong>, поэтому достаточно знания базовых тем для понимания исходного кода.
Загрузить шаблоны можно, нажав на кнопку "Code" и в ней "Download ZIP"
